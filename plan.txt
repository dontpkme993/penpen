1. 核心畫布與圖層系統 (Core Engine)
這是軟體的靈魂，決定了操作是否流暢以及專業度。

多圖層管理 (Layer Stack)： 支持圖層的新增、刪除、隱藏、鎖定、重新命名。

圖層混合模式 (Blending Modes)： 如正常、色彩增值 (Multiply)、濾色 (Screen)、覆蓋 (Overlay) 等。

非破壞性編輯： 支援圖層蒙版 (Layer Mask) 與調整層 (Adjustment Layers，如亮度/對比度)。

向量與點陣混合： 同時處理 SVG 向量路徑與 Bitmap 像素位圖。

畫布縮放與平移： 支援高頻率的縮放 (Zoom) 與平移 (Pan)，並具備參考線與對齊吸附功能。

2. 工具箱功能 (Toolbox)
定義使用者如何與圖像互動的基本手段。

選區工具 (Selection Tools)： 矩形、橢圓、套索，以及基於 AI 的「一鍵去背/主體選取」。

繪圖工具 (Painting Tools)： 筆刷 (Brush)、橡皮擦、漸層工具。筆刷需支援壓力感應（針對平板使用者）。

修補工具 (Retouching)： 仿製印章 (Clone Stamp)、污點修復、液化 (Liquify)。

文字系統： 支援系統字體與自定義字體上傳，具備字距、行距與文本框排版。

變形工具 (Transform)： 自由變形、旋轉、傾斜、透視校正。

3. 圖像處理與濾鏡 (Image Processing)
利用前端算力進行實時反饋。

基礎調整： 色階、曲線、色相/飽和度、色彩平衡。

高效能濾鏡： 高斯模糊、銳利化、雜訊消除。

AI 增強功能： 超分辨率 (Upscaling)、內容感知填充 (Content-Aware Fill)。

4. 技術架構與性能規格 (Technical Specs)
確保跨平台流暢度的關鍵字。

渲染引擎： 使用 Canvas 2D API 或 WebGL/WebGPU 進行硬體加速。

數據處理： 利用 WebAssembly (WASM) 處理運算密集的圖像處理邏輯（如 C++ 編寫的 OpenCV 模組）。

狀態管理： 完善的 Undo/Redo 歷史紀錄（基於 Command Pattern）。

文件支援：

導入：PSD, RAW, WebP, PNG, JPG, SVG。

導出：支持壓縮質量選擇、切片導出。

PWA 支持： 讓工具可以「安裝」到桌面，並在離線狀態下使用。

5. 使用者介面與體驗 (UI/UX)
工作區配置： 可自定義的面板佈局（如左側工具欄、右側屬性欄）。

快捷鍵系統： 盡可能相容 Photoshop 的原生快捷鍵（如 V 選取、B 筆刷）。

響應式設計： 自動適配電腦顯示器與 iPad Pro 等移動端設備。
